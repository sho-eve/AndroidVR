This is software from 2018.
Originally, it was designed to be used as a VRHMD by attaching various sensors to an Android terminal. However, many of the sensors have been discontinued, and since the operation cannot be confirmed, a set of VR assets have not been uploaded.
Just a reference to the formation of metaverse space and the application of Joy-con.
